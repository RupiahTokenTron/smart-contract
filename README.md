Mengapa  perlu Rupiah Protocol ( RPT )?
### 1.	Stable Value 
Rupiah Protocol ( RPT ) akan dipatok dengan nilai tetap 1 Rupiah Protocol
 ( RPT ) = 1 Rupiah ( IDR ). Rupiah Protocol ( RPT )  dapat digunakan sebagai media pertukaran dan penyimpan nilai di dunia blockchain. Nilai Rupiah Protocol ( RPT ) tidak akan di pengaruhi oleh harga asset lain seperti Bitcoin , Ethereum dan lain lain yang dapat berubah secara signifikan.
### 2.	Low Risk
Kami percaya ada dua kelompok orang yang membeli mata uang kripto hari ini. Kelompok kedua lebih peduli dengan aplikasi yang sebenarnya dari koin dan token tersebut, dan salah satu aplikasi paling penting adalah bahwa cryptocurrency membantu mereka untuk menukar dan menyimpan kekayaan tanpa pengaruh pasar. Rupiah Protocol ( RPT ) adalah cryptocurrcncy yang memiliki resiko rendah.
### 3.	Adoptions
Dengan pesat nya pertumbuhan Eonomi Digital khusunya pada sector keuangan dan pembayaran , Karena Rupiah Protocol ( RPT ) nilai nya tetap maka Rupiah Protocol ( RPT )  mata uang yang aman untuk di adopsi.
Apa itu Rupiah Protocol ( RPT )  
Rupiah Protocol ( RPT ) diciptakan di atas jaringan Tron Blockchain.
Rupiah Protocol ( RPT ) menjadi sistem keuangan terdesentralisasi dengan menjadi perbandingan utama untuk harga aset tertentu yang memungkinkan mekanisme keseimbangan untuk memastikan stabilitas nilai dari Rupiah Protocol ( RPT ) itu sendiri. 
Fase pengembangan Rupiah Protocol ( RPT )
Fase pertama disebut Fase Pembuatan, di mana Rupiah Protocol ( RPT ), token berbasis Tron Blockchain TRC20, dibuat. Rupiah Protocol ( RPT ) tidak terbatas dan memiliki mekanisme self-balancing. Rupiah Protocol ( RPT ) akan diperdagangkan di pasar cryptocurrency dan harganya akan stabil 1 Rupiah ( IDR ),
mencerminkan nilai utama stablecoin. Selain itu, selama Fase Pembuatan, tim manajemen dan komunitas kami akan berusaha untuk menyebarluaskan Rupiah Protocol ( RPT ) guna menambah pemegang Rupiah Protocol ( RPT ) dan membangun kerjasama yang kuat dengan perusahaan atau organisasi.
Fase ke dua adalah pengembangan, setelah Rupiah Protocol ( RPT ) diedarkan dan diterima secara luas, manajemen team akan meluncurkan Sistem Pembayaran berbasis Blockchain. Dimana pemegang Rupiah Protocol ( RPT ) dapat melakukan pembayaran dengan menggunakan Rupiah Protocol ( RPT ) di beberapa perusahaan rekanan Rupiah Protocol. Dan dapat menarik Rupiah Protocol ( RPT ) menjadi mata uang  fiat sesuai nilai dan jumlah Rupiah Protocol ( RPT ) yang di miliki.

## Perubahan Nilai Rupiah Protocol 
Nilai Rupiah Protocol ( RPT ) terhadap mata uang fiat dan cryptocurrency diatur oleh teori paritas daya pembelian dan lawan pertukaran. Nilai tukar Rupiah Protocol ( RPT ) di pengaruhi oleh nilai tukar Rupiah ( IDR ) terahadap mata uang lain.
Misalnya : 
1.	Emas ( T ) meliki harga 10 USD/gram di New York ( X ) dan 150.000 rupiah di indonesia ( Y ) sedangkan 15.000 adalah kurs Rupiah terhadap dollars (  Z  ). 
Berdasarkan paritas daya pembelian di atas maka :
Untuk membeli 10 gram emas dengan menggunakan Rupiah Protocol ( RPT ) adalah Jumlah T x X x Z = 10 x 10 x 15000 = 1500.000 ( RPT ) dengan kata lain jika di konversikan menjadi USD adalah jumlah RPT/Z = 1500.000/15000 = 100 USD
2.	Emas ( T ) meliki harga 10 USD/gram di New York ( X ) dan 130.000 rupiah di indonesia ( Y ) sedangkan 13.000 adalah kurs Rupiah terhadap dollars (  Z  ). 
     Berdasarkan paritas daya pembelian di atas maka :
Untuk membeli 10 gram emas dengan menggunakan Rupiah Protocol ( RPT ) adalah Jumlah T x X x Z = 10 x 10 x 13000 = 1300.000 ( RPT ) dengan kata lain jika di konversikan menjadi USD adalah jumlah RPT/Z = 1300.000/13000 = 115.4 USD
